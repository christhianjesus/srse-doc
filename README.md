# Pruebas

Las pruebas se realizan mediante el proyecto de Eclipse CDT que se encuentra
en la carpeta principal. Solo es necesario ejecutarlo y correrá todas las pruebas.

## Conjuntos de Prueba 1

Conjunto 1, cuenta con un total de 1000 fragmentos de código extraídos de la base
de datos de Stack Overflow. El script usado se encuentra en:
https://github.com/christhianjesus/srse-psql

## Conjuntos de Prueba 2

Conjunto 2, cuenta con un total de 195 fragmentos de código extraídos del libro Efficient
C Programming: A Practical Approach, cuyo autor es Mark Weiss, disponibles
en: https://users.cs.fiu.edu/~weiss/ecp/files.html

## Conjuntos de Prueba 3

Conjunto 3, cuenta con un total de 10 fragmentos de código extraídos de diversas
